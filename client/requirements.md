## Notes
- Mock JWT authentication: The client stores the token in `localStorage` and appends it to the `Authorization: Bearer <token>` header for all API requests.
- Images: Using high-quality Unsplash URLs for product seeding and hero banners to achieve the luxury editorial aesthetic.
- Fonts: Using 'Outfit' for display/headers and 'DM Sans' for body text to create a clean, modern, high-end look.
